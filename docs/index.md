# MulmoTerminal Guide

A browser-based terminal + GUI workspace for coding agents ([Claude Code](https://claude.com/claude-code)
and OpenAI's **Codex**). This guide focuses on the **grid view** — running many terminals side by side —
and how to configure it.

ブラウザで動く、コーディングエージェント（Claude Code / Codex）向けのターミナル + GUI ワークスペースです。
このガイドは **グリッドビュー**（複数ターミナルの並行実行）とその設定に絞って説明します。
