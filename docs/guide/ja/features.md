# 機能一覧
{: .no_toc }

グリッドビュー周辺の機能をまとめます。詳しい使い方は[基本編](basics.html)・[応用編](scenarios.html)へ。

- TOC
{:toc}

---

## グリッドとセル

| 機能 | 説明 |
|---|---|
| 並行ターミナル | 1 ページ最大 **9 セル**。あふれると**ページ（タブ）**が増える |
| セル追加 / 閉じる | `New terminal ＋` で追加、各セルの `✕` で閉じる |
| 並べ替え | 並べ替えモードで各セルの `◀ ▶` により位置を入れ替え |
| 拡大 / フィルムストリップ | `⤢` で 1 セルを拡大、他は下部にサムネイル。サムネイルのヘッダー余白クリックで切替 |
| 状態の色 | 作業中（青）/ 要対応（琥珀）/ アイドル（通常）を枠とドットで表示 |

## 起動方法（ランチャフォーム）

| 機能 | 説明 |
|---|---|
| Claude / Codex トグル | セルごとにエージェントを選択 |
| 作業ディレクトリ | 任意のディレクトリで起動。*cwd presets* で補完 |
| worktree 隔離 | git リポでタスク名を入れて worktree を作成・起動 |
| 起動コマンド | Claude 以外（`Shell` / `codex` / 任意コマンド）を**永続ターミナル**として起動 |
| スクリプト実行 | そのディレクトリの `script.json` のスクリプトをセル内で実行 |

## セッションの可視化

| 機能 | 説明 |
|---|---|
| git ステータスチップ | `⎇ ブランチ ●変更数 ↑ahead ↓behind` をヘッダーに常時表示 |
| モデル / コンテキスト量 | 例 `Opus · ctx 35%` — 動作モデルと文脈の埋まり具合 |
| コスト（推定） | 設定に **セッション / 今日 / 今月** の概算コスト |
| アクティビティタイムライン 🕘 | エージェントのツール呼び出し履歴（新しい順）をモーダルで表示 |
| worktree 差分バッジ | worktree セルで変更量を表示、クリックで差分パネル |

## コマンドセル（スクリプト実行）

| 機能 | 説明 |
|---|---|
| セル内実行 | `script.json` のコマンドを**そのセルの中で**実行（一時的、リロードで消える） |
| ✦ Summarize / Explain | 端末出力を `claude -p` に渡し、エラー/警告/原因/直し方を要約 |
| ⧉ Copy as prompt | コマンド + ディレクトリ + 要約 + フォローアップをコピーして任意セッションに貼付 |

## エージェント

| 機能 | 説明 |
|---|---|
| Claude Code | 既定のエージェント。フック駆動で作業中/要対応をリアルタイム表示 |
| Codex | Codex を第一級エージェントとして起動・再開。GUI パネルも Codex のツール呼び出しで駆動 |
| セッション永続化（tmux） | tmux があれば各セッションは tmux 内で走り、リロードやサーバ再起動を跨いで**再接続** |

## 見た目・ヘッダーのカスタマイズ

| 機能 | 説明 |
|---|---|
| 名前バッジ / 色 | `.mulmoterminal.json` でディレクトリごとに名前・ヘッダー/本体/枠/ドット/ボタンの色 |
| ヘッダーの操作ボタン | `buttons` で `input`（テキスト送信）/ `open`（URL・Finder・アプリ内ビュー）/ `shell`（コマンド実行）を追加 |
| ヘッダーの表示チップ | `chips` で組み込みチップの並べ替え・非表示 + カスタムチップ |
| テーマ | Midnight / Nord / Daylight / Solarized Light |

詳細は[設定方法](config.html)へ。
